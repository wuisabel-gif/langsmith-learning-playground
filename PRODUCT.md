# Product

## Register

product

## Users

Learners and developers exploring LLM agent observability — primarily the project's
author (an extension of the LangChain Academy "LangSmith Essentials" course) and anyone
they share it with. They sit down to *run* an agent graph, watch what it does step by step,
and understand traces/spans/evaluation hands-on.

## Product Purpose

A real, runnable lab: type a question, run an agent graph through OpenAI, and watch genuine
traces (spans, tokens, cost, latency, retrieval, LLM-judge) — instrumented with LangSmith.
Success = the concepts from the course "click" because you can see them happen live.

## Brand Personality

Playful, colorful, energetic — a learning *playground*, not a corporate console. Friendly
and approachable, but still credible as a real dev tool. Three words: vivid, lively, hands-on.

## Anti-references

- Generic SaaS-cream + slate dashboard (the template it started as).
- Sterile enterprise admin panels.
- Over-serious dark "hacker terminal" (considered and rejected for this one).

## Design Principles

1. **Show the run.** The agent's execution is the hero — make steps, states, and results visceral.
2. **Color carries meaning AND joy.** Each graph/step type gets its own hue; state changes glow.
3. **Real over decorative.** Every number/panel reflects an actual run; don't fake data.
4. **Approachable, not childish.** Playful energy that still reads as a competent tool.

## Accessibility & Inclusion

WCAG AA contrast for text on colored surfaces. All motion has a `prefers-reduced-motion`
fallback. Color is never the only signal (icons/labels accompany state).
